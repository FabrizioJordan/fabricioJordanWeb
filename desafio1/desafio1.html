--> En CSS, un breakpoint (punto de quiebre) se refiere a un valor específico de ancho de pantalla en el que se produce un cambio en la apariencia o el diseño de un sitio web. Los breakpoints se utilizan en el diseño web responsivo (responsive design) para adaptar la apariencia de un sitio a diferentes dispositivos y tamaños de pantalla.

--> Cuando se alcanza un breakpoint determinado, se aplican reglas CSS específicas que alteran la presentación del contenido para que se ajuste mejor al dispositivo o pantalla.

--> Los breakpoints se definen mediante consultas de medios (media queries) en CSS. Estas consultas permiten establecer condiciones basadas en el ancho de la pantalla o el dispositivo, y se utilizan para aplicar estilos personalizados en función de estas condiciones.

--> Al utilizar breakpoints en CSS, los diseñadores y desarrolladores web pueden crear diseños adaptables y optimizados para diferentes dispositivos, proporcionando una experiencia de usuario agradable en una amplia gama de pantallas.

--> Es importante mencionar que los breakpoints y el diseño responsivo en general son consideraciones importantes para garantizar la accesibilidad y usabilidad de un sitio web en dispositivos móviles y de escritorio.

--> En CSS, un media query (consulta de medios) es una técnica que permite aplicar estilos diferentes a un documento HTML en función de las características del dispositivo o medio en el que se visualiza. Permite adaptar la presentación de un sitio web a diferentes tamaños de pantalla, tipos de dispositivos y capacidades del usuario.

--> Los media queries se utilizan en conjunto con los breakpoints (puntos de quiebre) para crear diseños responsivos y adaptativos. Un media query consiste en una expresión que verifica ciertas condiciones, como el ancho de la pantalla, la resolución, la orientación, el tipo de dispositivo, entre otros.
